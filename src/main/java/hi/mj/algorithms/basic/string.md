## String 다루기

### String 클래스 기본 메서드
|   description         |   return  | method              |  
| --------------------- | -----------|--------------------|  
| 문자열 동일성 검사        | boolean  | equals(String)        |  
| 문자열 사전식 순서        | int       |comparerTo(String)   |  
| 문자열 길이             | int        | length()           |  
| 특정위치 문자            | char      |charAt(int)          |  
| 지정한 문자의 위치 검색    | int      |indexOf(char)          |  
| 지정된 범위의 부분 문자열   | String  | substring(int, int)   |  
|                        |         |                       |
